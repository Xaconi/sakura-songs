### Added
- **Sleep Timer**: Temporizador para detener la música automáticamente

  - Input para tiempo personalizado (1-480 minutos)
  - Countdown visible en badge flotante
  - Fade-out gradual de 5 segundos al finalizar
  - Mensaje "Dulces sueños" al completar
  - Modal responsive: bottom-sheet en móvil, flotante en desktop
  - Tests unitarios completos (56 tests nuevos)
